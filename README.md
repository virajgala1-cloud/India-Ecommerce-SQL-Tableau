
# India Ecommerce Sales Dashboard - TABLEAU.



# Project Overview
An end-to-end data analytics project analyzing Indian e-commerce sales data using MySQL and Tableau. The project involved writing SQL queries to extract insights from 3 relational tables and visualizing them in an interactive Tableau dashboard.


# Tools Used
 MySQL — Database creation, data import, SQL querying
 Tableau Public — Data visualization and dashboard creation.



 # Dataset
 Source: Kaggle
 3 tables: List of Orders (500 rows), Order Details (1500 rows), Sales Target (36 rows)
 Data covers Indian e-commerce orders across multiple states and cities.


 # SQL Concepts Used
 SELECT, FROM, WHERE
 JOINs (INNER JOIN across 3 tables)
 GROUP BY, ORDER BY
 Aggregate functions (SUM, COUNT, AVG)
 DATE_FORMAT, STR_TO_DATE
 LIMIT, DISTINCT


 # Key Insights
 Indore is the top performing city with ₹79,069 in total sales.
 January recorded the highest monthly revenue at ₹61,439.
 Clothing is the most profitable category with ₹11,163 profit.
 Ahmedabad shows negative profit despite decent sales volume.
 Printers and Sarees are the top selling sub-categories.
